# ML Portfolio: Wine Quality & Telco Churn Prediction

## 🚀 Projects Overview

This repository showcases two end-to-end Machine Learning projects completed as part of the AIML Data Science Accelerator at Noblex Infinity Labs.

### 1. 🍷 **Red Wine Quality Prediction (Vino Veritas)**
- **Objective**: Predict wine quality (3-8 scale) from physicochemical properties.
- **Key Models**: LightGBM (88.8% multi-class accuracy), XGBoost (90.4% binary)
- **Insights**: Alcohol, sulphates, and volatile acidity are top predictors.
- **Files**: [Wine Quality Notebooks](wine-quality/), [Project Report](Wine_Quality_Project_Report.docx)

### 2. 📞 **Telco Customer Churn Prediction**
- **Objective**: Predict which customers are likely to churn using customer data.
- **Key Model**: Random Forest (F1-Score 0.6374)
- **Insights**: Month-to-month contracts, short tenure, and lack of services drive churn.
- **Files**: [Telco Churn Notebook](telco-churn/), [Project Report](Telco_Churn_Project_Report.docx)

## 🛠️ Tech Stack
- Python, pandas, scikit-learn, XGBoost, LightGBM
- Jupyter Notebooks for EDA, modeling, and conclusions
- Model evaluation with hyperparameter tuning (GridSearchCV/RandomizedSearchCV)
