# 🍷 Red Wine Quality Prediction (Vino Veritas)

**Project Overview**

End-to-end ML analysis for predicting red wine quality based on 11 physicochemical properties.

- **Dataset**: 985 red wine samples
- **Models**: Classification (LightGBM 88.8%), Regression
- **Key Findings**: Alcohol (+0.51 corr), Sulphates, Volatile Acidity are top predictors
- **Recommendations**: Target ranges for winemakers to improve quality
