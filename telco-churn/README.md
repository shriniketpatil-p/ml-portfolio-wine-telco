# 📞 Telco Customer Churn Prediction

**Project Overview**

Machine Learning solution to predict and mitigate customer churn for a California telecom company (7,043 customers).

- **Best Model**: Random Forest (F1-Score 0.6374)
- **Key Drivers**: Month-to-month contracts, short tenure, no tech support
- **Business Value**: Targeted retention strategies with positive ROI
